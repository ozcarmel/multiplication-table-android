# Hebrew Store Listing

## App Name

לוח הכפל בקלות

## Short Description

לומדים ומתרגלים את לוח הכפל בצורה משחקית, בעברית ובאנגלית.

## Full Description

לוח הכפל בקלות היא אפליקציה לימודית לילדים שמאפשרת להבין ולתרגל את לוח הכפל בצורה נעימה, ברורה ומשחקית.

האפליקציה כוללת לוח כפל אינטראקטיבי 10x10, הסברים קצרים, קבוצות שוות, חיבור חוזר, ותרגול שבו הילד בוחר כפולות, מקליד תשובות, בודק אותן, מגלה תשובות בעת הצורך ומנקה את התרגול להתחלה חדשה.

האפליקציה תומכת בעברית ובאנגלית, כאשר עברית היא ברירת המחדל.

## Key Features

- לוח כפל אינטראקטיבי 10x10
- ברירת מחדל בעברית, תמיכה באנגלית
- הסבר קצר לכל תרגיל
- תרגול כפולות לפי סדר או בערבוב
- בדיקת תשובות מיידית
- כפתור גלה וכפתור נקה
- ללא התחברות
- ללא איסוף מידע אישי
- ללא פרסומות
