# English Store Listing

## App Name

Multiplication Made Easy

## Short Description

A playful multiplication table practice app for kids, with Hebrew and English support.

## Full Description

Multiplication Made Easy helps children understand and practice the multiplication table through a clear, playful, and interactive learning experience.

The app includes a 10x10 multiplication table, short explanations, equal groups, repeated addition, and a practice panel where children can choose a times table, type answers, check their work, reveal answers when needed, and clear the board for a fresh start.

Hebrew is the default language, and English is supported.

## Key Features

- Interactive 10x10 multiplication table
- Hebrew-first interface with English support
- Short explanation for each multiplication fact
- Practice by times table
- Ordered or mixed practice modes
- Instant answer checking
- Reveal and clear buttons
- No login required
- No personal data collection
- No ads
